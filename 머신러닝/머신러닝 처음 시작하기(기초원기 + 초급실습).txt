1) 머신러닝 이란 ? 
- 하드웨어적 발명을 머신러닝이 최근 핫해졌다.
- T/P/E가 각각 존재하는 것이 머신러닝이다.
- 단순 논문 다운은 머신러닝이 아니다 이것을 학습시켜야 머신러닝이다.

- 지도학습 : 분류 / 회귀 
input 데이터에 대해 올바은 데이터가 주어지는 것이 지도학습 
회귀 : 주가 예측이 대표적 예시 / 연속적인 값을 찾는 것 / 예측하는 것 
분류 : 미리정의된 클래스 레이블 중 하나를 예측 / 2진 분류와 다중 분류

- 비지도 학습 : 군집 / 시각화 
input데이터에 대해 올바른 정보가 주어지지 않을 경우 
군집 : 비슷한 것들 끼리 묶어 하나로 보이게 하는 것 
시각화 : 뒤에서 다시 다룬다 

- 준지도 학습 : 지도 학습 + 비지도 학습
페이스북 사진 속 얼굴을 알아본 다음 tag를 하는 것이 준지도 학습의 대표적 예시

- 강화학습 : 알파고에서 쓰인 방법이 바로 강화학습이다
올바른 학습 때 마다 보상을 줌 
특히 게임에 적용이 많이 됨 
스타 / 피카츄 배구 등등

2) Machine learning VS Programming 
Programming으로 하면 번거롭지만 Machine learning은 번거롭지 않다는 장점이 있다.
일반화 : Training데이터와 Input 데이터가 달라져도 똑같은 성능으로 출력하기 위한 목적 
-> 일반화가 제대로 이루어지지 않을 경우 문제 2가지 

- 과대적합 : 새로운 input데이터를 사용할 경우 정확도가 낮은 경우 / deep learning에서 주의해야함 
- 과소적합 : 조기종료 시점까지 충분히 학습 시키기
